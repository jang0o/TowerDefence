# Install Assistant

Install Assistant through the Package Manager to use it in the Unity Editor.

## Prerequisites

Before you install Assistant:

- Use Unity 6.0.60f1, Unity 6.3 (6000.3), or later.
- Accept the terms and conditions of the **AI** menu in the Unity Editor.

## Install the package

To install Assistant, follow these steps:

1. In the main menu, go to **Window** > **Package Manager**.
2. In the **Package Manager** window, select **Add** > **Add package by name**.
3. In **Name**, enter `com.unity.ai.assistant`.
4. Select **Add**.

   After installation, **Assistant** appears under **In Project** > **Packages - Unity**.

## Open Assistant

After installation, follow these steps to launch Assistant:

1. In the main menu, select **AI** > **Assistant**.
   The **Assistant** window opens.
2. (Optional) Dock the **Assistant** window in your layout to keep it accessible during your workflow.
